amq_broker

Installs and configures Apache ActiveMQ Artemis or Red Hat AMQ Broker services.

Dependencies

The roles depends on the redhat_csp_download role of middleware_automation.redhat_csp_download collection. To install, from the collection root directory, run:

ansible-galaxy collections install -r requirements.yml
